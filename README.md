# cpp-template<a id="sec-1"></a>

This is a basic template for my C++ projects.

## Building<a id="sec-1-1"></a>

```bash
git clone https://github.com/OrdoFlammae/cpp-template.git
cd cpp-template
mkdir build
cd build
cmake ..
make
./cpp-template
```
